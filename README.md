# Tol Eressea Vault

Personal reading archive powered by [Quartz](https://quartz.jzhao.xyz/).

Articles are saved via the [Tol Eressea Telegram bot](https://github.com/movewithjavier/tol-eressea) and automatically published to GitHub Pages.

## Live Site

https://movewithjavier.github.io/tol-eressea-vault

## Structure

```
content/
├── index.md              # Home page
└── articles/             # Saved articles
    └── index.md          # Articles index
```

## Development

```bash
npm install
npx quartz build --serve
```
